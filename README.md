# ESP8266-LED-Matrix-Clock

![Image](https://github.com/zangaby/ESP8266-LED-Matrix-Clock/blob/main/matrix_clock.png?raw=true)

Connections

MAX7219 to ESP8266

VCC - 3.3V

GND - GND

CS - D8

DIN - D7

CLK - D5

DS3231 to ESP8266

GND - GND

VCC - 3.3V

SDA - D1

SCL - D2
